# LLM-Previsor

Passos para execução

Executar comando para subir DB
docker compose up -d

Ativar Venv

Executar main.py

# Cria um ambiente virtual (opcional, mas recomendado)
python -m venv venv

# Ativar ambiente de desenvolvimento
.\venv\Scripts\activate

# Instala os pacotes (Caso esteja em venv, só ira instalar lá)
pip install -r requirements.txt

# Desativar venv
deactivate
